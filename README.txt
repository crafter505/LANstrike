LAN//STRIKE
================
Windows 原生局域网 FPS

操作
WASD：移动    鼠标：瞄准/转向    左键：攻击
1：手枪  2：战术刀  3：步枪  4：冲锋枪  5：霰弹枪  6：狙击枪
R：换弹    G：投掷触地瞬爆雷    Shift：冲刺    Tab：战绩
ESC：暂停菜单、灵敏度/全屏设置、回到主界面

地图会刷新绿色瞬爆雷补给，最多携带 3 枚。
LAN 使用 UDP 27020，主机首次运行时请允许 Windows 专用网络。

双击 outputs 里的“构建并运行 LANStrike.bat”重新生成 EXE。
